C++

C++ คือ ภาษา C programming language รุ่นใหม่ เป็นภาษาในการเขียนโปรแกรม ถูกพัฒนาโดย Dr.Bjarne Stroustrup ซึ่งเป็นนักวิจัยอยู่ทีห้องปฏิบัติการ Bell Labs ประเทศสหรัฐอเมริกาในระหว่างปี พ.ศ. 2525-2528 ภาษา C++ เกิดจากแนวคิดในการเพิ่มประสิทธิภาพภาษา CC โดยได้นำความสามารถของ ภาษา C มาพัฒนา ให้เป็นโปรแกรมภาษาที่มีความเป็น Object Oriented Programming (โปรแกรมเชิงวัตถุ) และนี้เองคือที่มาของภาษา C++ จากการพัฒนานี้ทำให้ทุกสิ่งที่ภาษา C ทำได้ ภาษา C++ ก็จะสามารถทำได้ แต่สิ่งที่ภาษา C++ ทำได้ ภาษา C อาจจะทำไม่ได้

        #include <iostream>
        using namespace std;
        int main() 
        {
          cout << "Hello, World!";
          return 0;
        }
        // Hello World
