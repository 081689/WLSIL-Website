PHP

PHP ย่อมาจากคำว่า “Personal Home Page Tool” (ปัจจุบันได้เพิ่มเติมคำย่อใหม่โดยรวมกับตัวย่อเป็น PHP : PHP Hypertext Preprocessor) ซึ่งเป็นภาษาประเภท Script Language ที่ทำงานแบบ Server Side Script กระบวนการทำงานจะทำงานแบบโปรแกรมแปลคำสั่ง interpreter คือแปลภาษาทุกครั้งที่มีคนเรียกสคริปต์ ข้อดีคือ ไม่ต้องนำไปประมวลผลใหม่ (Compiler) เมื่อจะนำโปรแกรมไปใช้งาน หรือจะอัพเดตเวอร์ชั่นของโปรแกรม สามารถอัพโหลดขึ้นไปทับไฟล์เดิมแล้วใช้งานได้ทันที ขอเสียที่ต่างกันอย่างชัดเจนก็คือ กรณี Syntax ผิดจะรู้ก็ต่อเมื่อมีผู้ใช้งานเจอบั๊ก

วิธีการเขียน Hello World

      
        <?php echo Hello World; ?>
        // Hello World

การเริ่มต้นการเขียน php จะเริ่มด้วยด้วย < โดยฟังก์ชั่นที่จะแสดงผลคือ echo และตามด้วยข้อความที่ต้องการแสดงผล

วิธีการคำนวนทางคณิตศาสตร์

      
        <?php echo 1 + 1; ?>
        // 2

การคำนวณทางคณิตศาสตร์ใน PHP จะใช้ตัวดำเนินการทางคณิตศาสตร์ได้หลายตัว ดยเครื่องหมานพื้นฐานได้แก่ + (บวก) - (ลบ) * (คูณ) / (หาร)

วิธีการใช้ตัวแปร

      
        <?php
          $name = "John Doe"
          $age = 24
          echo name + ", I'm " + age + "years old"; ?>
        # John Doe, I'm 24 years old

ตัวแปร (variable) คือชื่อหรือเครื่องหมายที่กำหนดขึ้นสำหรับใช้เก็บค่าในหน่วยความจำ ตัวแปรจะมีชื่อ (identifier) สำหรับใช้ในการอ้างถึงข้อมูลของมัน ในการเขียนโปรแกรม ค่าของตัวแปรสามารถที่จะกำหนดได้ใน run-time หรือเปลี่ยนแปลงอยู่ตลอดเวลาในขณะที่โปรแกรมทำงาน (executing)
ในตัวอย่าง เราได้ทำการประกาศ 2 ตัวแปร ในตัวแปร name เก็บค่าข้อความ John Doe และเป็นประเภทเป็น String ตัวแปร age เก็บค่า 24 และเป็นประเภทเป็น Integer และสุดท้ายทำการแสดงค่าโดยการรวบข้อความทั้งหมดเข้าด้วยกัน

แบบฝึกหัด

ให้นักเรียนเขียนโปรแกรมแสดงชื่อ นามสกุล และอายุ ในอีก 10 ปีข้างหน้า โดยใช้ความรู้ที่เรียนไป ตามตัวอย่างด้านล่างง 

      
       John Doe
       I'll be 25 years old in 2029
